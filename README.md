## 🚀 Como começar
Siga os passos abaixo para configurar o projeto em sua máquina local.
### 1. Pré-requisitos
Antes de começar, certifique-se de ter o **Node.js** instalado em seu computador. Recomendamos a versão **LTS (Long Term Support)**.
 * Verifique se o Node está instalado executando este comando no terminal:
   ```bash
   node -v
   
   ```
 * Se não estiver instalado, baixe a versão mais recente em nodejs.org.
### 2. Clonando o repositório
Após confirmar a instalação do Node.js, clone o repositório para sua máquina local:
```bash
# Clone o repositório
git clone "https://github.com/codariadev/projeto-estudo-main.git"

# Entre na pasta do projeto
cd nome-do-seu-projeto

# Instale as dependências
npm install
# ou yarn install / pnpm install / bun install

```
### 3. Rodando o projeto
Agora que o ambiente está configurado, inicie o servidor de desenvolvimento:
```bash
npm run dev
# ou yarn dev / pnpm dev / bun dev

```
Abra http://localhost:3000 no seu navegador para ver o resultado.
*Dica: Você pode substituir <URL_DO_SEU_REPOSITORIO> pelo link real quando o tiver em mãos.*
